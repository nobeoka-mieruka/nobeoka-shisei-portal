# 延岡市政見える化ポータル 共通開発方針

## プロジェクト概要

名称：
延岡市政見える化ポータル

公開URL：
https://nobeoka-shisei-portal.pages.dev/

目的：
延岡市民が、市長、市議会議員、議案、一般質問、財政、報酬、市役所案内などの公開情報を、スマートフォンから分かりやすく確認できるようにする。

## 編集方針

・本サイトは延岡市、延岡市議会の公式サイトではない
・公開資料を市民向けに整理した非公式サイトである
・特定の政党、会派、議員、候補者、政治団体を支持、推薦、批判しない
・推測、架空データ、根拠のない順位、独自採点を掲載しない
・確認できない情報は「確認中」「情報未登録」「公表なし」などで区別する
・出典、基準日、最終確認日を可能な限り表示する
・既存データを壊さない

## 主な情報源

・延岡市公式ホームページ
・延岡市議会公式ホームページ
・会議録
・議案書
・採決結果
・予算書、決算書
・選挙公報
・選挙管理委員会資料
・議員本人の公式ホームページ、公式SNS
・市長記者会見
・総務省などの公的機関資料

## 優先する機能

1. 市長公約達成状況
2. 議案ごとの賛否
3. 一般質問データベース
4. 財政ダッシュボード
5. 市役所案内診断
6. 議員一覧、議員詳細
7. 報酬比較
8. 更新履歴
9. サイト内検索

## 市長公約

・公式資料で確認できる範囲のみ掲載する
・独自の達成率や採点は原則使用しない
・情報が見つからないだけの場合は「未着手」にしない
・進捗状態は文字でも表示する
・公約原文、概要、進捗、根拠資料、出典、確認日を分けて管理する

## 議案・一般質問

・議案番号、件名、提出者、採決結果、賛否、出典を構造化する
・一般質問は議員、質問日、テーマ、質問内容、答弁、会議録リンクで整理する
・検索、カテゴリ絞り込み、議員別絞り込みを考慮する
・確認できない内容を補完しない

## 財政・報酬

・年度と基準日を明示する
・月額と年額を混同しない
・期末手当を含むか明記する
・全国順位や類似団体順位は、同一条件の公式個別データが揃った場合だけ掲載する
・推定順位は掲載しない

## アクセス解析

Google Analytics：
管理者向け分析として使用する。
測定IDは G-GHQCETJ7FN。
リアルタイム利用者、人気ページ、流入元、端末などを確認する。
React Routerのページ遷移でもpage_viewを送信し、二重送信を防ぐ。

Cloudflare Analytics：
サイト上に表示する累計アクセス数の取得元として使用する。
Google Analytics Data APIとは役割を分ける。
個人を特定できる情報はサイト上に表示しない。

## スマホ・アクセシビリティ

・スマートフォン表示を最優先する
・375px、390px、768px、1280pxを意識する
・横スクロールを発生させない
・文字が縦書きや不自然な改行にならないようにする
・タップ領域を十分確保する
・色だけで意味を伝えない
・キーボード操作とフォーカス表示に配慮する
・適切なaria-labelを使用する
・十分な文字コントラストを確保する

## データ管理

・大量のデータをReactコンポーネントへ直接書かない
・src/data配下のJSONまたはTypeScriptで管理する
・IDの重複を防ぐ
・不正な日付、URL、status、必須項目不足を検証する
・推測値や架空の日付を登録しない
・自動生成データと手入力データを分ける

## 品質確認

変更後は可能な限り次を実行する。

npm run validate:data
npm run typecheck
npm run lint
npm run build

validate:dataがない場合は、必要性を確認して提案する。

## Git・デプロイ

・秘密情報をGitHubへコミットしない
・.claude/settings.local.jsonなどローカル専用ファイルを原則コミットしない
・APIトークン、秘密鍵、環境変数の実値をコードへ書かない
・問題がなければmainへコミット、プッシュする
・Cloudflare PagesのGit連携による自動デプロイまで確認する

## 作業時の基本姿勢

・既存データや既存ページを壊さない
・重複実装を避ける
・必要以上に大きなライブラリを追加しない
・Lighthouse、Core Web Vitals、SEOを悪化させない
・ユーザーへの確認がなくても安全に進められる編集、検証、コミット、プッシュは自動で進める
・外部情報を調査する場合は公式一次資料を優先する
・判断できない内容は推測で実装せず報告する
