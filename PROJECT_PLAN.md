## プロジェクト概要

**名称**：延岡市政見える化ポータル
**公開URL**：https://nobeoka-shisei-portal.pages.dev/

**目的**：市長、市議会、議案、一般質問、財政、報酬、市役所案内などの公開情報を、市民がスマートフォンから分かりやすく確認できるようにする。

**基本方針**
- 公開資料のみ掲載
- 政治的中立
- 推測を書かない・架空データを書かない
- スマホ最優先

```
Excel／公式資料
      ↓
src/data 配下の JSON／TypeScript
      ↓
npm run validate:data （ID重複・不正日付・不正URL・必須項目チェック）
      ↓
npm run typecheck / lint / build
      ↓
GitHub（main）
      ↓
```
